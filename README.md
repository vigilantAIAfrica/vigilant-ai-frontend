Intuitive, offline-first interface for scam protection in African mobile money.

## Features
- Mobile-optimized UI for cheap Android phones.
- Swahili/Sheng localization.
- PWA support for offline access, aligning with SDG 9 (innovation).

## Tech Stack
- React.js or Flutter for cross-platform.
- Tailwind CSS for styling.
- Service Workers for offline.
- GitHub Pages for free hosting.

## Quick Start
1. Clone the repo: `git clone https://github.com/VigilantAI/vigilant-ai-frontend.git`
2. Install dependencies: `npm install`.
3. Start dev server: `npm start`.
4. Build: `npm run build`.
